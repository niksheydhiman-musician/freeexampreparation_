Q1 Design History Official Core

Which design philosophy pioneered the principle "Form Follows Function" at its foundational core?

💡 View Solution Architecture

✓ Correct Answer: B (The Bauhaus Movement)

Bauhaus unified structural function directly with artistic execution, stripping decorative excess to maximize product industrialization capacity.

Q2 Material Culture Official Core

The traditional technique of 'Eiko' or 'Ikat' weaving creates distinctive blurred patterns by employing which methodology?

💡 View Solution Architecture

✓ Correct Answer: B (Tie-dyeing the yarns prior to the weaving phase)

Ikat functions through a precise resist-dye system mapped directly onto warp or weft threads before individual pieces intersect on the physical loom equipment.

Q3 Color Informatics Official Core

When secondary colors are introduced through the mixing of two primary colors across a subtractive platform, what physical medium behavior is utilized?

💡 View Solution Architecture

✓ Correct Answer: B (Pigment absorption of selective light components)

Subtractive systems (such as CMYK or raw pigments) subtract white light wavelengths via physical absorption. The colors remaining unabsorbed represent the visible reflective palette.

Q4 Printing Systems Official Core

Which configuration defines the fundamental process variance of Serigraphy (Screen Printing) relative to Intaglio systems?

💡 View Solution Architecture

✓ Correct Answer: A (Ink is pressed directly through open stenciled apertures rather than recessed grooves)

Serigraphy functions using porous fabrics where mesh units are blocked using specific stencils, allowing ink flow to escape only onto designated coordinates underneath.

Q5 Spatial Logic Official Core

When mapping perspective parameters, a three-point perspective matrix introduces which dynamic that differentiates it from a two-point system?

💡 View Solution Architecture

✓ Correct Answer: B (A vertical vanishing point producing extreme bird’s-eye or worm’s-eye distortion)

Three-point fields eliminate vertical line parallelism, shifting vertical planes to converge at a dynamic structural point high above or deep below the main framework.

Methodology Analysis:

2. Set to equilibrium: 6(x² - 5x + 4) = 0 → (x - 4)(x - 1) = 0
